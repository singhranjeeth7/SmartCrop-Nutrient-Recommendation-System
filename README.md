# Smart Crop Nutrient Recommendation System

This precision agriculture solution helps farmers make informed nutrient management decisions using historical rainfall, soil fertility, and crop requirement data. It recommends optimal nutrients for specific crops and regions, improving yield while reducing environmental impact.

## Features
- Crop-wise nutrient recommendation
- Rainfall and soil fertility analysis
- Sustainable fertilizer usage

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Random Forest
